## 5 大流派特征 + 对标 prompt 模板

### 流派 1: 古风戏腔（45% 主流，京择主攻方向）

**音乐特征**：
- 调式：D/C 大调五声音阶
- 编曲：戏腔 hook（女高）+ 古筝 + 笛子 + 现代电子节拍
- 节奏：75-85 bpm
- 标志：副歌首句是戏腔（哎~~~~呀~~~~）

**MiniMax prompt 模板**：
```
Chinese opera style, soprano female vocal, guzheng, erhu, modern electronic beat 80bpm, dramatic hook, half-spoken intro
```

**歌词结构**（京择对标《牵丝戏》实战版）：
```
[Intro] 戏腔念白（哎~~~~呀~~~~）
[Verse 1] 现代叙事（4-6 句）
[Pre-Chorus] 短句铺垫（2-4 句）
[Chorus] 戏腔 + 现代对比（4 句，必有反转）
[Bridge] 现代口语 / 苦笑
[Final Chorus] 戏腔上行 + 解脱
[Outro] 戏腔 fade
```

**京择对标案例**：
- 半壶纱_Z（古风禅意）
- 牵丝戏_Z（古风戏腔）
- 琵琶行_Z（古风戏腔 + 古典 IP 改编）
- 明月照甲方（古风戏腔 + 京白反差）
- 知我者谓我加班（古风戏腔 + 典故密度）

### 流派 2: 古风禅意（15% 长尾，Z 世代新刚需）

**音乐特征**：
- 调式：C/D 大调五声音阶
- 编曲：古筝主奏 + 箫副奏 + 极简电子节拍
- 节奏：60-70 bpm（比戏腔慢）
- 标志：佛系/出世/治愈感

**MiniMax prompt 模板**：
```
ancient Chinese, guzheng, xiao flute, lo-fi chill, female vocal, 70bpm, meditation, Zen mood, half-念白
```

**京择对标方向**：
- "内卷退出指南"系列：Z 世代 5 分钟出离心（半壶纱_Z 首次实战）
- "佛系职场"系列：打工人的禅意日常
- "周末躺平"系列：工位禅意 2.0

### 流派 3: 国潮（10% 跨界，功夫/偶像/二次元）

**音乐特征**：
- 调式：五声音阶 + 西方 hip-hop 节拍
- 编曲：民乐管弦 + 808 bass
- 节奏：90-110 bpm
- 标志：武术/偶像/二次元题材

**MiniMax prompt 模板**：
```
Chinese pentatonic, hip-hop beat 95bpm, guzheng, erhu, modern trap, dramatic, traditional Chinese instruments
```

**京择对标方向**：
- "国潮十二时辰"系列（咏春/书法/茶道/武术/中医）
- "Z 世代武侠"系列

### 流派 4: 国风民谣（10% 大众化桥梁）

**音乐特征**：
- 调式：五声音阶 + 西方民谣吉他
- 编曲：木吉他 + 古筝 + 弦乐
- 节奏：70-80 bpm
- 标志：跨圈国民度（毛不易/一程山路）

**MiniMax prompt 模板**：
```
Chinese folk, acoustic guitar, guzheng, soft strings, 75bpm, indie pop, emotional, modern Chinese
```

**京择对标方向**：
- "国风民谣"系列（一程山路的现代版）
- 跨界国民度歌曲（适合发抖音/视频号）

### 流派 5: 京剧戏腔（5% 极致复古）

**音乐特征**：
- 调式：京剧西皮/二黄
- 编曲：京胡 + 京剧打击乐 + 现代元素极少
- 节奏：自由 / 无固定 bpm
- 标志：京剧唱腔 + 念白

**MiniMax prompt 模板**：
```
Chinese Peking opera vocal, jinghu, traditional Chinese percussion, dramatic, no modern elements, ancient
```

**京择对标方向**：
- 不建议（受众小，破圈难）
- 偶尔做"反差彩蛋"可以（如武家坡2021）

## 5 条对标建议

1. **古风戏腔 = 45% 主流**——`batch_gen_songs.py` 默认支持（pitfall #25 已固化 CLI 参数）
2. **古风禅意"出世感"=Z 世代新刚需**——可做"内卷退出指南"系列（半壶纱_Z 验证）
3. **古典 IP 现代改编 = 长青赛道**——李白/苏轼/白居易都有改编空间（琵琶行_Z 验证）
4. **戏腔 hook + 现代电子反差 = 短视频爆款公式**（牵丝戏_Z/明月照甲方验证）
5. **国风跨界（功夫/民谣/OST）=破圈流量入口**（不在主推，保留作系列扩展）
