## 2. 钩子公式 4 大模板

### 公式 A：**"脆弱 → 透光"型**（适合 港式 / 民谣 / 抒情）
```
[脆弱的词] 也敢 / 才能 / 正在 [动词+正能量]
```
- 例: "玻璃心 也敢透光" / "失败 也敢再试" / "碎过 才能完整"
- 套路: A→B 转折，让脆弱 = 力量
- 适用: 民谣 / Cantopop / 抒情

### 公式 B：**"沉重 → 轻盈"型**（适合 反差 / 电子 / 鸡汤）
```
[大词/沉重] 值得 / 配得上 [一个轻盈动作]
```
- 例: "活着就值得放一场烟火" / "加班 值得一杯奶茶" / "失意 配得上一场好觉"
- 套路: 把沉重事物绑定到具体小动作
- 适用: 8-bit / 电子 / Pop

### 公式 C：**"拟人化孤独"型**（适合 日常 / 城市民谣 / 文艺）
```
[具体场景+时间] 还在 [动词] 我
```
- 例: "凌晨四点的便利店 还在等我" / "末班地铁 还在等我" / "便利贴 还没掉"
- 套路: 把场景拟人化，"它"在等我 = 我不孤单
- 适用: 城市民谣 / Indie / Singer-songwriter

### 公式 D：**"半命题"型**（适合 怀旧 / 翻红 / 情感共鸣）
```
[我们/你/我] 后来 / 如今 都学会了 [新行为]
```
- 例: "后来的我们 都学会了不打扰" / "我们都学会了点赞" / "我们都学会了放下"
- 套路: 半命题，听众自动补完
- 适用: 怀旧 / 翻红 / 情感

## 4. 副歌上扬 5 个标准动作

| 动作 | 适用 | 例 |
|------|------|-----|
| **音高抬升** | 抒情 | 副歌高 2-3 度（晚风启程） |
| **能量翻倍** | 电子 | 第二遍加 8-bit arp 副旋律（活着） |
| **半念白 + 旋律** | 民谣 | 后半段 melody 替代念白（后来的我们） |
| **副歌 × 2** | 流行 | 第一次正常 / 第二次 soft + 第三次 lift（玻璃心） |
| **hook 重复 3 次** | 洗脑 | "凌晨四点的便利店 还在等我" × 3（凌晨便利店） |
