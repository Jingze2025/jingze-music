# 6 种歌词结构模板

> 京择式调性：**第一人称实战视角，技术洞察，非新闻搬运，口语化，有体感记忆**。
> 不写"我失去你"的悲剧，写"我还在路上"的轻盈。

## 模板 1: demo 直发（最对标郑润泽《12.31》）

**结构**：
```
[Intro] (半念白/英文 whisper)
[Verse] × 1 → [Pre-Chorus] → [Chorus]
[Verse] × 1 → [Pre-Chorus] → [Chorus]
[Bridge] → [Chorus - Soft]  ← 减弱副歌，差异化
[Outro] (半念白/英文 whisper)
```

**特征**：
- 钩子在副歌第一句（**4 秒抓人**）
- Bridge 必须有"观点反转"（"不是不努力/是慢慢来也算一种快"）
- Intro/Outro 用英文/半念白 制造留白感
- 主题锚定一个**具体时间点**（12.31 / 06.06 / 某个季节）

**字数**：600-1200 字
**实测**：1 次生成功（02.06《06.06》）

## 模板 2: 半念半唱（对标陈鸿宇/房东的猫）

**结构**：
```
[Intro] (全念白)
[Verse] × 1 (半念半唱) → [Verse] × 1 (纯唱) → [Chorus]
[Bridge] (全念白) → [Chorus - Half]
[Outro] (全念白)
```

**特征**：
- 主歌用"半念半唱"制造对话感
- 念白部分 ≤ 4 行 / 段
- 整体走"说书人"风格

---

## 模板 3: 古风（对标陈粒/银临）

**结构**：
```
[Intro] (古琴/箫前奏)
[Verse] (文言+白话混合) → [Pre-Chorus] → [Chorus]
[Bridge] (古诗词引用) → [Chorus]
[Outro] (留白)
```

**特征**：
- 押韵宽松，按意群
- 偶尔用"矣/乎/也"等文言虚词
- 意象：月/酒/剑/山水/古道

---

## 模板 4: 城市民谣（对标赵雷/陈鸿宇）

**结构**：
```
[Intro] (吉他前奏)
[Verse] × 1 → [Chorus]   ← 简单结构
[Verse] × 1 → [Chorus]
[Bridge] → [Chorus - Soft]
[Outro]
```

**特征**：
- 主歌直接叙事（地点+时间+场景）
- 副歌重复钩子 2-3 次
- 不用 Pre-Chorus（结构更短更利落）

---

## 模板 5: 电子/合成器流行（对标梓渝/邓紫棋）

**结构**：
```
[Intro] (电子前奏)
[Drop]   ← 第一波电子 hook
[Verse] × 1 → [Build Up] → [Chorus]
[Verse] × 1 → [Build Up] → [Chorus]
[Bridge] (synth solo) → [Drop 2]
[Outro]
```

**特征**：
- 2 个 Drop 制造电子爆点
- Build Up 用 8 拍递增
- 副歌可纯念白 + 节拍

---

## 模板 6: 纯音乐（无歌词）

**结构**（用 `is_instrumental: true`）：
```
[Intro]
[Theme A] (主旋律)
[Development] (展开)
[Theme B] (副旋律)
[Climax] (高潮)
[Resolution] (收束)
[Outro]
```

**特征**：
- 歌词字段省略
- 全部用 prompt 描述音乐走向
- 适合做视频 BGM / 公众号配乐

---

## 京择式写词 checklist

写完一首歌必走 5 条检查：

- [ ] **钩子在 4 秒内** — 副歌第一句可独立成为"金句"
- [ ] **意象具体** — "早班地铁" 比 "时光" 好；"备忘录写了又划掉" 比 "内心的挣扎" 好
- [ ] **不写大词** — 避免"孤独/梦想/未来"，改写"绿萝/相片/咖啡/朋友圈"
- [ ] **观点反转** — Bridge 必须有"非 A 是 B"的转折
- [ ] **半念白** — Intro/Outro 用半念白制造呼吸感（demo 直发风格最关键）

## 4 个对标分析维度

每次写词前，从对标歌曲里拆这 4 个：

1. **时间锚点** — 歌名/歌词是否含具体时间点？
2. **情感锚点** — 是"我失去你"还是"我还在路上"？或第三种？
3. **传播钩子** — 副歌第一句是什么？能在 4 秒内让人记住吗？
4. **结构模板** — 6 种选哪一种？

把这 4 点填完，歌词框架就出来了。
